#FICADICA: Para fazer uma imagem aparecer, usamos a tag <img>, lembra? A imagem a ser exibida, pode estar na mesma pasta do arquivo .html ou na internet, bastando indicar seu nome ou endereço no atributo src da tag <img>. Logo, para mudar a imagem de uma tag <img>, basta:

 a) Dar um identificador para a <img> 

 b) Para alterar sua imagem, basta alterar seu src. Ex: para alterar a imagem de <img id="foto_pessoa">, fazemos: foto_pessoa.src = 'alegre.jpg'. 

 c) Para fazer a imagem "sumir" basta alterar deixar seu src vazio. Ex: foto_pessoa.src = ''.
